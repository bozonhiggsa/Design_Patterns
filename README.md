# Examples of Design Patterns

Basic examples of design patterns.

Creational patterns:
- Abstract Factory;
- Factory Method;
- Builder;
- Singleton (a number of versions);
- Prototype (a number of versions).

Behavioral patterns:
- Observer;
- Command;
- Chain of Responsibility;
- Mediator;
- Template Method;
- Strategy;
- Iterator;
- State;
- Visitor.

Structural patterns:
- Composite;
- Proxy;
- Decorator;
- Adapter.

Technologies:
- Java 8;
- Logback Classic.

Prerequisites:
- Better use the Maven 3;
- Better use the JDK 1.8_x.
