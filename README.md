# Examples of Design Patterns

Basic examples of design patterns.

Creational patterns:
- Abstract Factory;
- Factory Method;
- Builder;
- Singleton (a number of versions);
- Prototype (a number of versions).

Behavioral patterns:
- Observer;
- Command;
- Chain of Responsibility;
- Mediator;
- Template Method;
- Strategy;
- Iterator;
- State;
- Visitor.

Structural patterns:
- Composite.

Technologies:
- Java 8;
- Logback Classic.

Prerequisites:
- Maven 3 or 2;
- Better use the version of IntelliJ IDEA, JDK 1.8_x.
